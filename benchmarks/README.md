

Files
-----

 - `Makefile`     - Makefile for `g++` to compile `cppsurfops.so`.
 - `benchmark.py` - Runs and compares performance between various Python and C++ surface operations (invoke with `python ./benchmark.py` after building `cpusurfops.so` by calling `make`).

 - `benchmarklist.py` - List of Python functions in `pysurfops.py` and C++ functions in `cppsurfops.so` to use.
 - `cppsurfops.cpp` - C++ functions to modify a Pygame surface in various ways.
 - `pysurfops.py` - Python functions to modify a Pygame surface in various ways.
